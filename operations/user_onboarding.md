# 用户引导流程设计

## 注册流程设计

### 流程概述
```
访问首页 → 注册账号 → 邮箱验证 → 设置密码 → 完成注册 → 首次登录引导
```

### 注册方式
1. **邮箱注册**（推荐）
   - 输入邮箱 → 发送验证码 → 输入验证码 → 设置密码 → 完成
   - 预计耗时: 1-2分钟

2. **手机号注册**
   - 输入手机号 → 发送短信验证码 → 输入验证码 → 设置密码 → 完成
   - 预计耗时: 1-2分钟

3. **第三方登录**
   - 微信登录 / 支付宝登录 / Apple ID
   - 一键授权，快速进入
   - 预计耗时: 30秒

### 注册激励
- 新用户注册即送 **7天高级会员**
- 完成首次股票添加再送 **3天高级会员**

## 引导教程

### 第一步：欢迎引导（首次登录）

**弹窗标题**: "欢迎使用智选股"

**核心信息**:
1. 产品核心价值介绍
2. 三步上手指南
3. "开始体验"按钮

**引导内容**:
```
┌─────────────────────────────────────┐
│     欢迎使用智选股 v1.0             │
│                                     │
│  📊 让投资更聪明                    │
│                                     │
│  3步上手：                         │
│  ① 添加自选股票                    │
│  ② 查看智能分析                    │
│  ③ 设置提醒通知                    │
│                                     │
│  [开始体验]  [跳过引导]            │
└─────────────────────────────────────┘
```

### 第二步：功能导览

**覆盖区域**: 页面顶部导航

**引导内容**:
- 首页 → 查看市场概览
- 选股 → 发现优质股票
- 持仓 → 管理投资组合
- 消息 → 获取智能提醒

### 第三步：实操教程

**教程1：添加自选股**
1. 点击"添加股票"按钮
2. 输入股票代码或名称
3. 选择添加至自选
4. 完成

**教程2：查看分析报告**
1. 点击任意自选股票
2. 查看基本面、技术面、消息面分析
3. 点击"智能诊断"获取AI分析

**教程3：设置提醒**
1. 进入股票详情页
2. 点击"设置提醒"
3. 选择提醒条件（价格、涨跌幅、公告等）
4. 保存设置

## 首次使用体验优化

### 用户旅程地图

| 阶段 | 用户行为 | 优化措施 |
|------|----------|----------|
| 访问 | 首次访问首页 | 首屏展示核心价值，CTA按钮突出 |
| 注册 | 完成账号注册 | 流程简化，支持第三方快速登录 |
| 引导 | 完成新手引导 | 分步引导，避免信息过载 |
| 首次使用 | 添加第一只股票 | 默认推荐热门股票，降低决策成本 |
| 留存 | 持续使用 | 推送个性化分析报告 |

### 关键体验优化点

1. **减少摩擦**
   - 注册流程不超过3步
   - 支持游客模式体验核心功能
   - 清除不必要的输入项

2. **即时价值**
   - 注册后立即展示数据分析
   - 提供默认自选股组合
   - 首次分析报告免费开放

3. **渐进式引导**
   - 新功能逐步解锁提示
   - 工具提示（Tooltip）随用随学
   - 视频教程可随时查看

4. **成就系统**
   - 完成首次添加股票 → "入门达人"徽章
   - 完成首次设置提醒 → "风控先锋"徽章
   - 连续使用7天 → "坚持不懈"徽章

### 首次使用检查清单

- [ ] 注册/登录流程顺畅
- [ ] 新手引导清晰易懂
- [ ] 核心功能可快速上手
- [ ] 数据展示直观专业
- [ ] 界面操作流畅无卡顿
- [ ] 价值感知明确

## 引导流程技术实现

### 前端组件
- **引导弹窗（OnboardingModal）**: 全屏/半屏引导层
- **功能指引（FeatureTour）**: 高亮功能区域
- **工具提示（Tooltip）**: 悬浮提示信息
- **进度指示器（ProgressIndicator）**: 引导步骤进度

### 埋点事件
- `onboarding_start`: 引导开始
- `onboarding_step_complete`: 引导步骤完成
- `onboarding_skip`: 引导跳过
- `onboarding_complete`: 引导完成
- `feature_first_use`: 功能首次使用
