# 支持文档

## 常见问题（FAQ）

### 账号与安全

**Q1: 如何注册账号？**
A: 支持邮箱、手机号、微信/支付宝/Apple ID注册。推荐使用微信一键登录，最快30秒完成。

**Q2: 忘记密码怎么办？**
A: 点击登录页"忘记密码"，输入注册邮箱或手机号，根据指引重置密码。

**Q3: 如何修改个人资料？**
A: 登录后点击右上角头像 → "个人中心" → "资料编辑"。

**Q4: 账号安全如何保障？**
A: 我们采用银行级加密技术，支持两步验证（2FA），登录异常会实时预警。

### 功能与使用

**Q5: 免费版和付费版有什么区别？**

| 功能 | 免费版 | 高级会员 |
|------|--------|----------|
| 自选股票数 | 10只 | 无限 |
| AI智能诊断 | 3次/天 | 无限 |
| 技术指标分析 | 基础5个 | 全部50+ |
| 消息面监控 | 延迟30分钟 | 实时 |
| 风险预警 | 不支持 | 支持 |
| 专属客服 | 不支持 | 支持 |

**Q6: 如何添加自选股票？**
A: 首页点击"添加股票"，输入股票代码/名称/拼音首字母搜索，点击添加。

**Q7: AI智能诊断准确吗？**
A: AI基于大数据分析和机器学习模型，提供参考建议。股市有风险，投资需谨慎。

**Q8: 如何设置股价提醒？**
A: 进入股票详情页 → 点击"设置提醒" → 选择条件（价格/涨跌幅/公告）→ 保存。

**Q9: 数据更新频率是多少？**
A: 实时行情延迟3-5秒，财报数据T+1更新，新闻资讯实时推送。

**Q10: 支持哪些股票市场？**
A: 目前支持A股（上海、深圳）、港股、美股核心标的。

### 订阅与支付

**Q11: 如何订阅高级会员？**
A: 点击首页"升级会员"或"个人中心" → "会员中心" → 选择套餐 → 支付。

**Q12: 支持哪些支付方式？**
A: 微信支付、支付宝、银行卡、Apple Pay。

**Q13: 会员可以退款吗？**
A: 7天内未使用高级功能可申请全额退款，超过7天不支持退款。

**Q14: 会员到期后数据会丢失吗？**
A: 不会。历史数据永久保留，降级后仅限制新功能使用。

---

### 技术问题

**Q15: App闪退怎么办？**
A: 尝试以下步骤：
1. 清除缓存后重试
2. 更新至最新版本
3. 卸载重装

**Q16: 数据加载慢怎么办？**
A: 检查网络连接，或尝试切换网络（WiFi/4G）。

**Q17: 为什么看不到某只股票？**
A: 可能是以下原因：
- 股票不在支持范围内
- 代码输入错误
- 已退市股票

## 使用教程

### 新手入门

#### 1. 快速开始
```
步骤1: 下载安装App/访问官网
步骤2: 注册账号（推荐微信登录）
步骤3: 完成新手引导
步骤4: 添加自选股票
步骤5: 查看智能分析
```

#### 2. 核心功能教程

**教程一：自选股管理**
- 添加：搜索框输入代码/名称
- 排序：按涨幅/跌幅/时间排序
- 分组：创建自定义分组（如"潜力股"、"长线持有"）
- 删除：左滑股票→删除

**教程二：智能诊断**
- 进入股票详情页
- 点击"AI诊断"按钮
- 查看综合评分和建议
- 查看详细分析报告

**教程三：技术分析**
- 进入股票详情页
- 切换"技术面"标签
- 选择指标（MACD/KDJ/布林线等）
- 查看买卖点信号

**教程四：消息监控**
- 进入股票详情页
- 点击"消息面"标签
- 订阅关键公告提醒
- 设置自定义筛选条件

### 进阶技巧

#### 1. 选股策略
- **价值投资**: 筛选低PE、高ROE股票
- **成长投资**: 筛选高营收增速股票
- **趋势交易**: 筛选站上均线的股票

#### 2. 组合分析
- 创建多个自选分组
- 对比不同行业表现
- 跟踪调仓动态

#### 3. 风险控制
- 设置止损提醒
- 关注基本面恶化信号
- 分散投资提醒
