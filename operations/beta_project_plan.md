## 📊 反馈收集

### 反馈维度

| 维度 | 问题示例 |
|------|---------|
| **易用性** | 界面是否清晰？操作是否顺畅？ |
| **功能性** | 功能是否满足需求？缺少什么？ |
| **性能** | 加载速度如何？是否卡顿？ |
| **准确性** | 数据是否准确？ |
| **美观度** | 界面设计是否好看？ |

### 反馈形式

```
1. 每日打卡 (1分钟)
   - 今天使用了哪些功能？
   - 有没有问题？
   - 满意度评分 (1-5星)

2. 周度反馈 (10分钟)
   - 本周使用总结
   - 最喜欢的功能
   - 最需要改进的地方
   - 具体建议

3. Bug报告 (随时)
   - 问题描述
   - 重现步骤
   - 截图/录屏
   - 期望结果
```
